About

Web Programing and Designer

Hello! I am a computer engineering student and a technology-enthusiast web designer. I focus on designing user-friendly and aesthetic web pages by combining the analytical thinking and problem-solving skills I gained during my computer engineering education with my creative side. I develop projects using my knowledge in both software and design.

I believe that aesthetics and functionality should be balanced in web design. For this reason, I also attach importance to UX/UI principles as well as technologies such as HTML, CSS, JavaScript. My goal is to create modern and responsive websites that will take the user experience to the highest level.

As someone who constantly improves myself, I follow innovations and developing technologies closely. I aim to provide user-oriented, efficient and effective solutions in the web design and development process. Using technology in a way that adds value to human life is a great passion for me.

Expertise

My Expertise Areas

I am an expert in web design and development. Designing user-friendly and aesthetically appealing websites and optimizing performance using modern technologies are among my priorities. While trying to maximize user experience with UX/UI design, I can contribute to every stage of the projects with my full-stack development competence. In addition, I create mobile-friendly and responsive designs that work seamlessly with all devices, ensuring that users have an uninterrupted experience on every platform.

Web Design and Development

I specialize in designing modern and responsive websites with user experience at the forefront. I develop aesthetic and functional web pages using technologies such as HTML, CSS and JavaScript. My goal is to create user-friendly interfaces and optimize the speed and performance of the site.

User Experience (UX) and User Interface (UI) Design

I attach great importance to user experience (UX) and user interface (UI) design in web projects. I shape my designs according to the needs of users and ensure that they navigate the web page easily and pleasantly. I offer user-focused, intuitive and effective designs.

Full-Stack Web Development

I have both front-end and back-end development skills. I am competent in developing dynamic and database-supported web applications with JavaScript and various web frameworks (React, Node.js, etc.). I can contribute to a wide range of projects from database management to server installation.

Responsive and Mobile-Friendly Designs

I specialize in responsive web designs that adapt to mobile devices and different screen sizes. I provide a seamless experience to users by ensuring that my designs work best on all devices. I create flexible and scalable designs with techniques such as media queries and CSS grids.
