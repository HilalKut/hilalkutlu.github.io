# Hilal Kutlu Portfolio Website

Bu repo, Hilal Kutlu'nun kişisel portfolyo web sitesini içerir.
Site; modern, responsive ve erişilebilir bir yapıda hazırlanmış statik bir GitHub Pages projesidir.

## Canlı Site

- [https://hilalkut.github.io/hilalkutlu.github.io/](https://hilalkut.github.io/hilalkutlu.github.io/)

## İçerik

- `index.html`: Ana sayfa (hero, yetkinlikler, öne çıkan proje, iletişim)
- `about.html`: Hakkımda, eğitim/deneyim, kariyer hedefleri
- `projects.html`: Proje detayları ve case-study bilgileri
- `style.css`: Tüm görsel sistem ve responsive stiller
- `script.js`: Menü, aktif link, animasyon ve form etkileşimleri
- `robots.txt` ve `sitemap.xml`: SEO için temel indeksleme dosyaları

## Kullanılan Teknolojiler

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages

## Öne Çıkan Özellikler

- Mobil uyumlu modern tasarım
- SEO ve sosyal medya meta etiketleri
- Erişilebilirlik geliştirmeleri (skip link, focus stilleri)
- Scroll reveal animasyonları
- İletişim CTA ve form geri bildirim akışı

## Yerel Çalıştırma

1. Repoyu klonlayın:
   `git clone https://github.com/HilalKut/hilalkutlu.github.io.git`
2. Klasöre girin:
   `cd hilalkutlu.github.io`
3. `index.html` dosyasını tarayıcıda açın.

## Yayınlama

Bu repo `main` branch üzerinden GitHub Pages ile yayınlanır.
`main` branch'ine gönderilen güncellemeler kısa süre içinde canlıya yansır.
